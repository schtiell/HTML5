Los elementos en HTML 5 trabajan con modelos de caja, de CSS, los cuales utilizan el 100% del ancho de la pagina, es por eso que al establecer un color de fondo con la propiedad background se rellena una especie de caja.

Margenes en HTML 5:

Mediante las hojas de estilo CSS podemos modificar los márgenes de una página web. Esto hará que nuestro contenido se posicione en un lugar u otro de la página. Lo podremos pegar al borde superior, separarlo de la izquierda,... Todo aquello que necesitemos para darle estilo a la página web. Si queremos modificar los márgenes de una página web deberemos de utilizar cuatro propiedades CSS:

margin-bottom, para el margen inferior.
margin-left, para el margen izquierdo.
margin-right, para el margen derecho.
margin-top, para el margen superior.

Padding

La propiedad CSS padding establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde ( border ) no se permiten valores negativos.
